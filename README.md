## 💚 Implementation List

- 타이머
  - 한국 표준시 기준
  - 한국 시간 타이머
  - 미국 시간 타이머
- 입력
  - 입력 데이터의 형식: 숫자(1,2,3,4...)
  - 잘못된 형식의 입력데이터는 예외 처리
- 시작
  - 버튼을 누르면 소팅 시작
- 결과
  - 결과 데이터 형식: 숫자(1,2,3,4...)
  - 오름차순 필드
  - 내림차순 필드
  - 오름차순 바로 출력, 3초 후에 내림차순 출력
